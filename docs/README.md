# ImgToPDF - 이미지 PDF 병합 도구

여러 개의 이미지 파일(JPG, PNG)을 하나의 고화질 PDF로 쉽고 빠르게 병합해주는 데스크톱 애플리케이션입니다. `img2pdf` 라이브러리를 사용하여 이미지의 화질 저하 없이 원본 품질 그대로 PDF를 생성합니다.

## ✨ 주요 기능

- **고화질 병합**: 이미지를 재인코딩하지 않아 화질 손실이 전혀 없습니다.
- **다양한 입력 방식**:
  - 버튼을 통한 폴더 또는 개별 파일 선택
  - 드래그앤드롭(Drag & Drop) 지원으로 간편하게 파일 추가
  - 하위 폴더 포함 옵션 지원
- **출력 경로 지정**: 기본값(`[원본]/result`) 외에 사용자가 원하는 저장 위치 지정 가능
- **스마트 정렬**: 파일 이름 내의 숫자를 인식하여 사람이 인지하는 순서대로(Natural Sort) 자동 정렬
- **편의 기능**: PDF 생성 완료 후 즉시 저장 폴더 기능을 열 수 있는 버튼 제공
- **모던 UI**: 깔끔하고 직관적인 인터페이스

## 🚀 시작하기

### 설치 및 실행 (개발자용)

1. 저장소를 클론합니다.
2. 가상환경을 생성하고 활성화합니다.
3. 필요한 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
4. 프로그램을 실행합니다:
   ```bash
   python ImgToPDF.py
   ```

### 실행 파일(EXE) 빌드

PyInstaller를 사용하여 단일 실행 파일을 만들 수 있습니다:
```bash
pyinstaller --clean ImgToPDF.spec
```
빌드가 완료되면 `dist` 폴더 안에 `ImgToPDF.exe`가 생성됩니다.

## 🛠 기술 스택

- **언어**: Python 3.x
- **GUI**: Tkinter (ttk)
- **PDF 변환**: `img2pdf`
- **이미지 처리**: `Pillow`
- **드래그앤드롭**: `tkinterdnd2`
