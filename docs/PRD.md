# PRD – ImgToPDF (v0.1.0)

## 1. 개요
ImgToPDF는 여러 개의 이미지 파일(JPG, PNG)을 하나의 PDF 파일로 병합하는 데스크톱 GUI 애플리케이션이다.  
본 제품은 **화질 저하 최소화**를 최우선 목표로 하며, `img2pdf` 라이브러리를 활용하여 이미지 재인코딩을 방지한다.

## 2. 목표
- 이미지 품질 저하 없이 PDF 병합
- 비개발자도 사용 가능한 직관적인 GUI 제공
- Windows 환경에서 단일 실행 파일(EXE) 배포

## 3. 대상 사용자
- 스캔 이미지, 사진을 PDF로 정리하려는 일반 사용자
- 개발 지식 없이 빠르게 PDF를 생성하려는 사무/개인 사용자

## 4. 주요 기능 요구사항

### 4.1 입력
- JPG, JPEG, PNG 파일 지원
- 입력 방식
  - 폴더 선택
  - 파일 다중 선택
  - 드래그앤드롭(파일/폴더)
- 하위 폴더 포함 옵션

### 4.2 처리
- Natural Sort 기반 파일 정렬
- img2pdf 기반 PDF 병합
- 페이지 크기 = 이미지 크기 (리사이즈 없음)
- 고해상도 이미지(40MP 이상) 경고 표시

### 4.3 출력
- 기본 출력 경로: `[source]/result`
- 사용자 지정 출력 경로 지원
- 출력 파일명: `merged_YYYYMMDD_HHMMSS.pdf`
- PDF 생성 완료 후 결과 폴더 열기

## 5. 비기능 요구사항
- OS: Windows 우선 지원
- 성능: 수백 장 이미지 처리 가능
- 안정성: 오류 발생 시 사용자에게 명확한 메시지 제공
- 배포: PyInstaller 기반 단일 exe

## 6. 제약사항
- 이미지 편집(회전/리사이즈)은 범위 외
- PDF 옵션(A4 고정 등)은 향후 확장 기능
